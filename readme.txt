Fresh Tomatoes

This script consists of a class movie which generates Information about movies
that are supplied by entertainment_center.py. fresh_tomatoes.py provides the
HTML and CSS scripts that render the movie database into an browser-supported
HTML-File.

Installation

Download the files entertainment_center.py, media.py, fresh_tomatoes.py

Open Terminal in the file directory

$ python entertainment_center.py

the browser will open by itself loading the generated HTML-File
